Sincroniza productos de Discogs con tu tienda WooCommerce de forma automática.

== Description ==

Discogs Sync es un plugin que te permite importar productos de Discogs directamente a tu tienda WooCommerce. Con este plugin podrás:

* Buscar productos en la base de datos de Discogs
* Importar información detallada de productos (título, descripción, imágenes, etc.)
* Sincronizar automáticamente con tu inventario de WooCommerce
* Gestionar licencias y límites de uso

== Installation ==

1. Sube el plugin a la carpeta `/wp-content/plugins/discogs-sync/`
2. Activa el plugin desde el panel de administración de WordPress
3. Ve a Configuración > Discogs Sync
4. Configura tu URL de API, clave de licencia y dominio
5. ¡Comienza a importar productos!
